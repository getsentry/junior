# CV and Claim Verification

Use this reference to validate public resume/profile claims against public evidence.

## Scope

- Extract key claims from CV/resume/public profiles.
- Corroborate each high-impact claim using independent public artifacts.

## What counts as evidence

- Matching GitHub contributions (repos, commits, PRs, releases).
- Public talks or published technical writing that aligns with claims.
- Public package ownership or maintainer history.
- Public employment/project records when explicitly stated.

## What does not count by itself

- Personal references, endorsements, or recommendations (for example LinkedIn recommendations).
- Uncorroborated self-descriptions.

These may be noted, but must not drive scoring without artifact-level corroboration.

## Output requirements

- `CV and claim verification`: include `evidence sufficiency`, `verified claims`, `unverified claims`, and `References`.
- `verified claims`: include claim + corroborating source(s).
- `unverified claims`: include claim + missing evidence note.
